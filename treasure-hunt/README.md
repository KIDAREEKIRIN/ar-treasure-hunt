# 💎 보물 탐험 AR (Demo B → 게임 v4)

그 자리에서 몸을 돌려 360°에 숨은 보물을 찾는 WebAR 게임입니다. 마커 없이 자이로 기반이며,
스마트폰·태블릿 브라우저에서 링크로 실행됩니다.

## 게임 방식
1. **난이도 선택**(쉬움/보통/어려움) → 보물 수·지뢰·제한시간·웨이브가 달라집니다.
2. **3·2·1 카운트다운** 후 시작.
3. 화면 아래 **노란 화살표**가 가장 가까운 진짜 보물 방향을 가리킵니다(위/아래 ▲▼ 힌트 포함).
4. 조준점을 보물에 **0.65초** 맞추면 획득.
5. 제한시간 안에 모든 진짜 보물을 모으면 **클리어**, 시간이 0이 되면 **게임오버**.

| 난이도 | 보물 | 지뢰 | 시간 | 웨이브 |
|--------|------|------|------|--------|
| 쉬움   | 6    | 0    | 90초 | 단일 |
| 보통   | 9    | 2/웨이브 | 70초 | 12초마다 |
| 어려움 | 12   | 3/웨이브 | 55초 | 10초마다 |

## 보물 종류 · 시간 보너스 · 웨이브
- 🟡🔵🟣 **종류별 점수 차등**: 일반 1점 · 희귀 3점(파랑 글로우) · 전설 5점(보라 글로우+오라 링). 최종 점수 = **종류 점수 × 콤보**. 등장 비율은 `TYPES`의 `weight`로 조절.
- 🟢 **시간 보너스 아이템**: 주우면 제한시간 +7초. 점수엔 안 들어가고 **콤보도 깨지 않습니다**. (레이더는 안내하지 않으니 탐험 중 발견하는 보너스)
- 🟥 **지뢰**: 시간 −5초, 점수 −1, 콤보 초기화.
- 🌊 **웨이브**: 일정 시간마다(또는 화면의 진짜 보물을 다 모으면) 남은 보물을 **재배치**하고 새 보물·지뢰·시간템을 보충 → 한 영역을 털어도 계속 신선하게.

## 콤보 · 최고기록 · 파티클 · 사운드
- 🔥 **콤보 보너스**: 3초 안에 연속으로 보물을 모으면 콤보가 쌓이고, **콤보 수가 곧 획득 점수**가 됩니다(x1→x2→x3…). 지뢰를 밟으면 콤보 0으로 초기화.
- 🏅 **최고기록 저장**: 난이도별 최고 점수를 `localStorage`에 저장. 시작 화면 버튼과 종료 화면에 표시되고, 갱신 시 "신기록!".
- ✨ **파티클**: 보물 획득 시 색상 파티클이 터집니다(지뢰는 붉게).
- 🔊 **사운드 강화**: 콤보가 높을수록 음이 올라가는 획득음, 지뢰 폭발음(노이즈+저음), 카운트다운 비프, 승/패 징글 — 전부 WebAudio 신스(외부 파일 없음).

## 보물 3D 모델(.glb) — 종류별 다른 모델
- 종류마다 다른 .glb 사용: **`models/common.glb` · `rare.glb` · `epic.glb`** 를 넣으면 자동 적용.
- 로드 우선순위(종류별): **로컬 파일 → 네트워크 샘플 → 색 도형**(자동 폴백).
- **어떤 크기의 모델이든 자동으로 크기·중심 보정**(`autoFit`) — 스케일 조정 불필요.
- 무료 모델 출처와 적용법은 [`models/README.md`](models/README.md) 참고.
- 설정은 `index.html` 상단 `TYPES[].models` / `CONFIG.modelTargetSize`.

## 콤보 게이지 UI
- 콤보가 1 이상이면 HUD 아래 **게이지 막대**가 콤보 유지시간(3초)이 줄어드는 만큼 줄고, 색이 **초록→노랑→빨강**으로 변합니다. 비면 콤보가 끊깁니다.

## 실행 방법 (카메라는 HTTPS 필수)
```bash
cd treasure-hunt
python3 -m http.server 8000
# 다른 기기(아이패드/폰)에서 테스트하려면 HTTPS 터널이 필요:
ngrok http 8000     # 출력된 https://… 주소를 기기 브라우저로 열기
```
또는 Netlify/Vercel에 폴더를 드롭하면 즉시 HTTPS URL. `file://` 더블클릭은 카메라가 막혀 동작하지 않습니다.

## 손쉽게 조절 (index.html 상단)
- `LEVELS` — 난이도별 보물 수(`count`)/웨이브당 보충량(`perWave`)/지뢰(`fakes`)/시간템(`timeItems`)/제한시간(`time`)/웨이브 간격(`waveMs`)/배치 반경
- `TYPES` — 보물 종류별 `points`·`weight`(등장비율)·`tint`·`sizeMul`·`aura`
- `CONFIG` — `modelUrls`·`modelTargetSize`·`fakePenaltySec`·`timeBonusSec`·`comboWindowMs`
